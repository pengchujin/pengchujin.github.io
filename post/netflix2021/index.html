如何在中国使用 Netflix｜Netflix 入门指南🧭

2021-06-05 13 min read # 数字难民

Netflix 是世界上最大的流媒体，在全球除了中国大陆、克里米亚、伊朗、朝鲜、叙利亚外的所有国家地区都有提供服务。我个人从 2017 年就一直有订阅 Netflix，Netflix 有很全的中文字幕，在内容、画质、以及使用体验也都远超国内平台。希望你能认真看完这篇文章，应该能帮助你进入 Netflix 的世界。

🚫 突破网络限制

（本篇内容没有任何推广、利益成分，请放心；如果你有更好的推荐也可以在评论里指出；下面很多网站可能需要你使用代理才能打开）

第一步：我们要解决网络问题，因为你在中国大陆 Netflix 并没有在这里提供服务，我们需要使用代理软件通过其它国家地区的网络来访问观看 Netflix。有很多朋友可能使用的是 VPN，比如 ExpressVPN 、NordVPN 等，但这些 VPN 类不适合大陆的用户，我更推荐你使用「✈️机场」，这些网站所提供的代理更适合在大陆使用，也是绝大多数在中国大陆观看 Netflix 朋友的选择。

推荐的「✈️机场」排名不分先后，注册购买套餐后都会有使用说明文档教你在电脑、手机、路由器客户端上使用。

  • AmyTelecom
  • AAEX（需要挂代理打开）
  • BlinkLoad（需要挂代理打开）
  • Dler Cloud（需要挂代理打开）
  • Nextitally（需要挂代理打开）

如果你对这些很感兴趣你可以在 毒药机场评测 看到更多，选择更多。

除了「✈️机场」这种付费直接给你提供代理的方式，如果动手能力强，你当然可以选择自己购买海外的 VPS（云服务器），然后搭建自己的代理。

💰 Netflix 如何自己订阅｜合租

自己订阅 Netflix

  • 外币信用卡 （信用卡上印有 VISA、mastercard、American Express）

如果没有外币信用卡注册全球付这类虚拟信用卡也可以，更多虚拟信用卡的内容可以看看 VPS大玩家的推荐，基本都是存钱花适合对外币信用卡不放心的朋友。

Netflix 是订阅制按月收费，只有付费订阅的用户才能登录观看。如上图 Netflix 香港地区的套餐分为三个标准：基本、标准、和高级，只有高级套餐支持 4K 画质和 4 个屏幕同时观看，所以我更推荐你购买 Netflix 高级套餐。

不同地区订阅价格不同

这是份 Netflix 全球各地区价格 的谷歌表格有每一个国家地区的套餐订阅价格，最低和最高订阅的价格足足差了四倍价格。你订阅付款的区不会影响你观看的内容，你观看到内容是由你正在使用的 IP 决定。（这个后面还会更详细的介绍）

推荐你注册香港（因为比较方便，限制也少，价格稍贵一点）巴西或者巴基斯坦（很便宜，但你得有这个两个国家的代理，脸黑的可能还会让你验证本地手机号码）；土耳其价格是最低的但已经限制香港 IP 播放观看，并且注册付款审核很严格，所以不是很推荐。

订阅过程

挂好你想订阅付款区的代理，打开 netflix.com 官网（这里以香港地区为例）。

输入邮箱和密码，就可以选择套餐（推荐选择高级），这个时候显示的货币和价格是由你的代理决定。

最后输入你信用卡的信息即可。

订阅完成后，你可以绑定你的手机号码也可以选择跳过（如果是低价区可能会出现强制验证当地手机号码）。

最后会让你添加 profile 子账户，最多可以 5 个，这也是为什 Netflix 可以合租或者分享给朋友一起用。5 个人使用同一个账号密码登录，登录后可以选择自己名字的 profile 子账户，浏览观看记录都是分开的，首页也会有不同的推荐。均摊下订阅费用，即使是价格稍贵的港区人均也才十几元。

5 个子账户 profile 你可以自己选择是否是儿童账户，并没有 4 个成人账户 1 个儿童这种规定。至于合租应该是 4 个人还是应该 5 个人你可以自己考量。因为有最多 4 个屏幕同时在线观看的限制，但事实是只要大家都只自己使用，几乎不会触碰到这种限制。

作为乘客直接购买合租车位

如果你觉得信用卡有点麻烦，你也可以直接付钱购买合租提供的子账户。有国人提供了平台，他们已经订阅注册好了低价区的账户，然后再出租子账户给你。这种方简单直接支付宝微信付款就行，坏处是和你合租的人你并不知道是谁，大家都用一个账号体验好坏就靠大家自觉，价格也比你自己订阅稍微贵一点。

  • 奈飞 pro（合租平台，不含推广）
  • 奈飞小铺（合租平台，不含推广）

如果你想更低的价格，你也可以看看一些由个人提供的自己的车位合租。比如 逮虾户，或一些 telegram 合租平台。价格会低一些，但是靠谱不靠谱得自己辨别下（尽量别一下直接交年费，避免被骗损失过多）。

如果你只是想体验一个月尝试下，淘宝、拼多多直接搜索也会有「月抛车」类型，非常方便省事。

📼 如何在设备上观看 Netflix

在所有设备上 Netflix 都是不能调整在线播放的画质的，Netflix 会根据你的网络速度来给你提供你能流畅观看的最高画质，所以不要抱怨 Netflix 的画质很糊，你应该考虑你的网络代理速度是不是足够快。

iPhone iPad 等 iOS 设备

在国内的 App Store 你是搜索不到 Netflix 的，你需要注册一个非国区的 Apple ID 登录 App Store 才能下载这个 App。🤌荔枝数码教程

在手机上打开代理，打开 Netflix App 登录账户就能正常观看了。在 iPhone 8 以及更新的 iPhone 上你都能享受 HD 1080p 高清和 Dolby Vision 杜比视界的 HDR。如果你用的是 iPad Pro，使用 Safari 浏览器你甚至能播放 4K 超高清的内容（App 目前仍然限制 1080p）。

Android 手机平板

很多 Android 手机即使是在 Google Play 上也是搜不到 Netflix App，这是因为你的设备并没有通过 Google 的认证。不过没关系，Netflix 并不依赖 Google 框架，你可以直接在 APKPure 下载 Netflix 安装即可。

国内很多手机厂商并没有做 DRM L1 认证，没有 DRM L1 认证的 Android 观看 Netflix 时都会限制在 SD 540p 的画质。你可以在 Netfllix 官网查看 Android 设备 HD、HDR 的支持状况，FotixChiang 也有整理成 Google 文档，有很多国产手机、电视盒子对 Netflix 的支持状况。

Windows

Windows 推荐你使用 Edge 浏览器 和 Windows 商店里的 UWP 版本 Netflix App（你需要在设置里将国家地区切为美国才能搜索到）这两者都能支持到 4K HDR，前提你的设备符合 Netflix 这些要求。

在 Windows Edge 和 App 里使用 「Ctrl + Alt + Shift + D」可以调用显示出 Netflix 实时的画质等播放信息。推荐几个插件在 Edge 上都能够使用：

  1. Netflix 多语言字幕
  2. IMDB 评分

macOS

macOS 推荐你使用自带的 Safari 浏览器，最高也是能够支持到 4k HDR 的，你可以参看 Netflix 文档看下自己的电脑设备支持的情况。

使用 「Control + Shift + Option + D」 可以调用显示出 Netflix 实时的画质等播放信息。

通过电视、投影仪等大屏幕观看 Netflix 肯定能获得最佳的体验，但前提你得整好家里的网络环境，你的路由器网络要能挂上代理才能让电视、电视盒子等设备直接就能连上 Netflix。这里就不细说，你 Google 搜索下面的关键词就能看到很多教程，也可以在 恩山论坛找找资源资料。

电视、电视盒子客户端

绝大部分电视或者你家的电视盒子是安装不了 Netflix 客户端的，只有认证的过才可以。Sony 最近几年出的电视可以直接下载 Netflix App 安装到电视使用，使用 LG、三星的电视可以参看Bohao 的换区教程后安装使用 Netflix。

如果你的电视本身不支持 Netflix，你也可以通过购买有 Netflix 的电视盒子，这里推荐几个，性价比都很高都支持 4K HDR。

  1. Amazon FireTV Stick 4k（又称作火棒，淘宝售价 300+）
  2. Google ChromeCast With Google TV（淘宝售价 400+）
  3. Apple TV 6 代（淘宝售价 1400+）

游戏主机

很高兴的告诉你，你家的大型路由器 XBox、Sony PS 游戏主机系列都有 Netflix 认证是可以安装使用 Netflix 的。

Netflix 每个国家地区内容都不同，你的 IP 决定你看到的内容🔓

什么是 Netflix 解锁

在以前 Netflix 会禁止大部分代理播放观看以及下载视频，一些“漏网之鱼”没被 Netflix 标记为代理的就会被称为「原生IP/节点」或者「Netflix解锁 IP/节点」。

现在 Netflix 基本不会禁止代理观看内容，只是这些 IP/节点 限制只能看 Netflix 自制 originals 的内容，因为这些内容 Netflix 都拥有全球范围的版权；如果是 Netflix 解锁的 IP/节点 你就能看到更多本地化的内容，最直接的标志是你打开 Netflix 能看到国家地区的排行。（极少国家地区可能不会出现排行榜，你可以根据是否能看到封面不带 「N」 Netflix logo 判断）

切换节点观看不同内容库的 Netflix

举个例子，在美区 Netflix 是没有《老友记》的，你把代理切换到台湾、日本等国家地区就能在 Netflix 观看《老友记》了。unogs.com 这个宝藏提供在线搜索，你可以查看到你想看的电影剧集在哪个国家地区有，然后将代理切换到那个国家地区就可以了（前提你的 节点/IP 是解锁了 Netflix 的）。

各个国家地区的内容库数量和偏好都不同，全球整体来说 Netflix 拥有的内容确实非常多。大陆用户推荐你使用 台湾、香港、新加坡解锁节点观看，几乎每个内容都是有中文字幕的，并且整体的内容偏好和大陆也很接近。

✨ 一些经验技巧

Netflix 一般都在周五下午3点或4点更新整季内容

选在周五自然是大多数人都在周五开始休息（TGIF - thanks god is friday），而基本都是整季剧集更新也是 Netflix 的特色，让大家能真正意义上的刷剧。

Netflix 自制内容上新基本都是 PDT 太平洋时区凌晨（Netflix 总部洛杉矶的时区），对应国内的时间是下午3点（夏令时）或者4点（冬令时）。（美国和加拿大原本于每年10月的最后一个星期日凌晨2时起实施冬令时间；4月的第一个星期日凌晨2时起，恢复夏令时间）

Netflix 会在 YouTube 提前放预告片

Netflix 亚洲 和 Netflix 频道 在 YouTube 都会很频繁的更新影视剧集的上新预告片等内容，国内的影视号的大多数 Netflix 相关内容也都来自于这些地方。

Netflix 技术博客 会分享 Netflix 在技术层面的做的努力，如果你是国内相关视频平台的人员，我强烈建议你关注，认真学习，谢谢了。

Netflix 新闻编辑室 更多则会发布一些合作规划和一些影视剧集被砍还是开拍等规划安排。

Netflix Twitter 也有很多号会分享和 Netflix 相关新闻讯息内容。

在网页版你可以设定 Netflix 的语言，头像，以子账户访问密码

在右上角选择 管理使用者 就能修改头像，昵称，语言（语言会体现在你所有客户端上），是否自动播放以及年龄分级。

在右上角选择账户，你还可以给子账户设置 使用者锁定，在和不熟悉的人共享 Netflix 账户，能一定程度保护你的隐私。

安全性问题

  • Netflix 账户的所有者是对应这个邮箱的拥有者，你可以更改密码，邮箱拥有者可以通过邮件找回。

  • Netflix 邮箱可以修改或者说赠送给他人，前提是旧的邮箱收到邮件确认。

  • 子账户 profile 的共享形式不是特别安全，推荐熟悉的朋友一起共享使用。

  • Netflix 订阅付款区是可以修改的，你只用取消停止订阅并到期以后，挂目标区的代理打开重新订阅即可。
